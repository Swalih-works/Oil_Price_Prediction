# Oil Price Prediction

## Overview

Oil prices are influenced by various economic, geopolitical, and market factors, making accurate forecasting a challenging task. This project analyzes historical crude oil price data and develops predictive models to forecast future prices, helping businesses and consumers make informed decisions.

## Problem Statement

Oil price fluctuations have a significant impact on industries, transportation costs, and economic planning. The objective of this project is to identify historical trends and patterns in crude oil prices and build a machine learning model capable of predicting future price movements.

## Features

- Historical crude oil price analysis
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering for improved model performance
- Predictive modeling and forecasting
- Model evaluation using regression metrics
- Data visualization for trend analysis

## Key Insights

- Identified long-term trends and seasonal patterns in crude oil prices.
- Analyzed historical price fluctuations and market behavior.
- Generated forecasts to support data-driven decision-making.

## Model Evaluation

The model performance was evaluated using:

- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)
- R² Score

## Results

The developed forecasting models successfully captured underlying trends in crude oil prices and generated reliable predictions based on historical market data.
